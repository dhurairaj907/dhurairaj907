# Projects

## Personal Finance Manager

One of my most recent and largest projects is an AI-powered Personal Finance Manager built using React, TypeScript, Express, PostgreSQL, and Google Gemini.

The goal of this project was to create more than just an expense tracker. I wanted to build a complete financial companion capable of helping users manage their money, track savings, and receive personalized financial insights through AI.

### Features

* User Authentication and Session Management
* Income and Expense Tracking
* Saving Goals Management
* Goal Progress Monitoring
* Account Deletion
* AI Financial Assistant powered by Gemini
* Multi-turn Chat Memory
* Welcome Email System
* Goal Completion Email Notifications
* Cloud Database using Neon PostgreSQL
* Production Deployment using Render
* Responsive UI and Dashboard

### Technologies Used

* React
* TypeScript
* Express
* PostgreSQL
* Drizzle ORM
* Google Gemini API
* Resend API
* Express Sessions
* Render
* Neon Database

### Skills Acquired

Working on this project helped me gain practical experience in:

* Full-stack application development
* REST API design
* Session-based authentication
* Database schema design
* Cloud database management
* AI integration with contextual conversations
* Email service integration
* Production deployment and debugging
* Environment variable management
* State management using React Query
* Error handling and API validation
* Maintaining chat history and conversational context

This project also taught me how to solve real deployment issues, manage sessions with PostgreSQL, and integrate external APIs into a production-ready application.

## Indoor Air Quality Monitoring System

This is one of my favorite IoT projects.

The project uses an Arduino Uno along with MQ135 and DHT11 sensors to measure air quality, temperature, and humidity in real time. Sensor readings are synchronized with Firebase and displayed through a React-based web dashboard.

### Technologies Used

* Arduino Uno
* MQ135 Sensor
* DHT11 Sensor
* Firebase
* React

### Skills Acquired

* Sensor interfacing
* Real-time data collection
* Cloud database integration
* Hardware-software communication
* Dashboard development
* IoT architecture design

Working on this project helped me understand how embedded systems, cloud services, and web applications can work together as a complete system.
